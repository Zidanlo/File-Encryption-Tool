## وصف المشروع

**File Encryption Tool** هو أداة آمنة وسهلة الاستخدام لتشفير وفك تشفير الملفات باستخدام خوارزمية Fernet (مبنية على AES-128)، مع دعم كلمات مرور قوية مشتقة من PBKDF2. تم بناء الواجهة باستخدام Streamlit لتوفير تجربة مستخدم تفاعلية وبسيطة، مما يجعلها مثالية للمبتدئين والخبراء في مجال الأمان الرقمي. هذا المشروع مفتوح المصدر ومصمم لأغراض تعليمية وأمنية مشروعة فقط.

### الميزات الرئيسية
- **تشفير آمن**: استخدام Fernet لتشفير الملفات بطريقة غير قابلة للكسر بدون كلمة المرور.
- **فك تشفير موثوق**: استرجاع الملفات الأصلية بسهولة باستخدام كلمة المرور الصحيحة.
- **كلمات مرور قوية**: دعم توليد مفاتيح من كلمات مرور المستخدم مع ملح عشوائي (salt) لتعزيز الأمان.
- **واجهة بسيطة**: Dashboard تفاعلية مع Streamlit، تدعم رفع الملفات وتنزيل النتائج مباشرة.
- **دعم متعدد الأنواع**: يعمل مع أي نوع ملف (صور، نصوص، فيديوهات، إلخ).
- **أمان إضافي**: لا يتم حفظ أي بيانات على الخادم؛ كل العمليات محلية.

## متطلبات النظام
- **Python**: الإصدار 3.8 أو أحدث.
- **مكتبات مطلوبة**: Streamlit, Cryptography.
- **نظام التشغيل**: متوافق مع Windows, macOS, Linux.
- **مساحة تخزين**: لا توجد متطلبات خاصة، لكن الملفات الكبيرة قد تحتاج إلى ذاكرة RAM كافية.

## التثبيت
1. **استنساخ المشروع**:
   ```bash
   git clone https://github.com/yourusername/File-Encryption-Tool.git
   cd File-Encryption-Tool
   :
تثبيت المتطلبات:


pip install -r requirements.txt

تشغيل تطبيق:
python3 main.py

سيفتح التطبيق تلقائياً في متصفحك على http://localhost:8501.
طريقة استخدام :
تشغيل الـ Dashboard: بعد التثبيت، شغل الأمر أعلاه وانتقل إلى الرابط المعروض.
تشفير ملف:
اختر علامة التبويب "تشفير ملف".
ارفع الملف المطلوب.
أدخل كلمة مرور قوية (8 أحرف على الأقل، مزيج من الأحرف الكبيرة/الصغيرة، الأرقام، والرموز).
اضغط "شفر الملف" ونزل الملف المشفر.
فك تشفير ملف:
اختر علامة التبويب "فك تشفير ملف".
ارفع الملف المشفر (.encrypted).
أدخل كلمة المرور نفسها.
اضغط "فك تشفير الملف" ونزل الملف الأصلي.
نصائح استخدام :
أمان كلمة المرور: استخدم مدير كلمات مرور لتخزينها بأمان. لا تنسها، فلا يمكن استرجاع الملفات بدونها.
حجم الملفات: مناسب للملفات الصغيرة إلى المتوسطة؛ للملفات الكبيرة، قد يستغرق وقتاً أطول.
اختبار: جرب مع ملف نصي صغير أولاً للتأكد من العمل.
